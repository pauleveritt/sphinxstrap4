Goals
=====

- Fully embrace modern frontend development for HTML/CSS, specifically
  Bootstrap 4, SASS, and HTML 5

- Have HTML/CSS artifacts that don't have a hundred errors/warnings

- Works seamlessly with ABlog

- Simplify and reduce the surface area of customization

    - Tap into SASS for most CSS customization
